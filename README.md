# 🌸 Rashi Portfolio

A modern and responsive developer portfolio showcasing my projects, skills, education, and experience.

## 🚀 Live Demo
🔗 [https://your-portfolio-link.com](https://rashiportfoliocz.netlify.app/)

## ✨ Features
- 👩‍💻 About Me
- 🛠️ Technical Skills
- 🚀 Featured Projects
- 🎓 Education & Certifications
- 💼 Experience
- 📱 Fully Responsive Design
- ⚡ Smooth Animations

## 🛠️ Tech Stack
- React
- TypeScript
- Tailwind CSS
- TanStack Start
- TanStack Router
- TanStack Query
- Vite
- Radix UI
- Lucide React

## 💻 Run Locally

```bash
git clone https://github.com/rashiuwu/rashi-portfolio.git
cd rashi-portfolio
npm install
npm run dev
```
